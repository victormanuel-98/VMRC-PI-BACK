# Proyecto Intermodular – Sprint 4

## Descripción
Este repositorio contiene el trabajo del **Sprint 4** del módulo de Proyecto Intermodular (DAW).  
El objetivo del sprint fue diseñar el **modelo de dominio y base de datos**, la **API** y descomponer las **HUs en tareas técnicas** para Jira Cloud.

## Entregables del sprint
- **DER/UML:** imagen + fuente editable (.drawio).  
- **Especificación API:** `openapi.yaml`, guía de errores (`errores.md`) y convenciones (`convenciones.md`).  
- **Tablero Jira Cloud:** HU1.1 a HU1.5 desglosadas en tareas técnicas.

## Notas
- La base de datos es **conceptual**, basada en el DER.  
- La API está diseñada siguiendo **enfoque API‑first**, con endpoints, validaciones y contratos definidos.  
- Todas las tareas técnicas están planificadas y organizadas por capas: Datos, Servicios/API, Integración y Pruebas.
