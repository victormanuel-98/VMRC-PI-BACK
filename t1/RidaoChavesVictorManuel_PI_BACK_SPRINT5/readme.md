# SPRINT 5: MODELADO PLANTUML

## Descripción
Este repositorio contiene el BackEnd de la aplicación **Calculadora de Calorías**:
- API REST para gestión de usuarios, alimentos, recetas y comidas.
- Validación y almacenamiento de datos en la base de datos.
- Endpoints para consumir desde FrontEnd.

---

## Tecnologías
- Base de datos SQL / NoSQL
- PlantUML para diagramas de modelado (Sprint 5)
